# Airline Pricing Dashboard

This Power BI dashboard analyses airline ticket pricing trends using DAX and interactive visualisations.

## Key Insights
- Prices increase significantly as departure dates approach.
- Business class fares are substantially higher than Economy.
- Last-minute bookings show the highest average ticket prices.
- Ticket prices vary across airlines and number of stops.
